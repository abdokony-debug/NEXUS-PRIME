# Kony Marketing System

Automated marketing system for finding and contacting potential buyers.

## Setup

1. Clone repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and fill in values
4. Run: `npm start`

## Configuration

Edit `.env` file for settings:
- Google Sheets API credentials
- Campaign mode and batch size
- Target region and platforms

## Usage

Run manually:
```bash
npm start
